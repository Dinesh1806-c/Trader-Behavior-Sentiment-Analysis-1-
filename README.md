# Trader Behavior vs Market Sentiment Analysis

## Project Overview

This project analyzes the relationship between **Bitcoin market sentiment (Fear & Greed Index)** and **trader performance** using historical trading data.

The goal is to identify how market sentiment influences trader profitability and trading behavior.

## Dataset

Two datasets were used:

### 1. Bitcoin Fear & Greed Index

Columns:

* Date
* Classification (Fear / Greed)

### 2. Historical Trader Data

Columns include:

* Account
* Coin
* Execution Price
* Size USD
* Side
* Closed PnL
* Timestamp

## Project Workflow

1. Data Loading
2. Data Cleaning
3. Date Processing
4. Dataset Merging
5. Trader Performance Analysis
6. Data Visualization
7. Insights and Conclusions

## Key Insights

* Traders tend to perform better during **Greed market sentiment**.
* **Sell trades increase during Fear periods**, indicating defensive trading behavior.
* A small group of traders contributes significantly to overall profits.
* Market sentiment strongly influences trading decisions.

## Output

The analysis provides insights into how **market psychology impacts trading performance**, which can help traders improve their strategies.
